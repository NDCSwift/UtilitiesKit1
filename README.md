# 🛠️ UtilitiesKit1

A Swift Package providing a collection of utility extensions and helpers for iOS development.

## 📖 What this is
This Swift Package contains reusable utility functions and extensions that can be dropped into any iOS project. It's structured as a Swift Package Manager library, making it easy to integrate as a dependency. A practical toolkit for common tasks you'd otherwise have to rewrite in every project.

## ✅ Why you'd use it
- **Reusable utilities in one place** — common helpers you'd write yourself, already written
- **Swift Package Manager ready** — add it as a dependency in any Xcode project or package
- **Lightweight and focused** — no bloat, just practical extensions and functions

## 🚀 Getting Started
1. Clone the repo
2. Open `Package.swift` in Xcode (or `swift build` from terminal)
3. Add as a local or remote Swift Package dependency in your project
4. Import the module and use the utilities

## 📝 Notes
- Swift Package — no `.xcodeproj` file
- Requires Xcode 14+

## ⚙️ Requirements
- iOS 16+
- Xcode 14+
- Swift 5.7+
